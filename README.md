# Sprite OC - Real-time Pose Detection Interface

ウェブカメラと姿勢検出AIを組み合わせた、インタラクティブなスプライト操作体験を提供するプロジェクトです。

## 概要

このプロジェクトは、ブラウザ上で動作するインタラクティブなアート/ゲーム体験です。バックエンドの姿勢検出サーバーから送られてくる座標データを受け取り、p5.js と p5.play を使用して、ユーザーの動きに合わせたアニメーション（スプライト）を表示します。

## 主な機能

- **リアルタイム姿勢検出**: 手首、肩、鼻などのポジションをリアルタイムで追跡。
- **インタラクティブモード**:
    - **Mode 1 (Run)**: ユーザーの動きに合わせて逃げたり追いかけてきたりするキャラクター（Reeps）。
    - **Mode 2 (Rainy)**: 右手に「葉っぱの傘」が出現し、降り注ぐ雨から身を守る体験。
    - **Mode 3 (Elecy)**: 歯車やパイプに触れると電気が走ったり、電球が灯ったりする工場風ギミック。
- **管理ダッシュボード (`Untitled-2.html`)**:
    - 姿勢検出の状態確認、FPS表示、カメラ映像のプレビュー、フルスクリーン切り替えなどが可能。

## 技術スタック

- **Frontend**: HTML5, CSS3, JavaScript
- **Graphics & Animation**: [p5.js](https://p5js.org/), [p5.play](https://p5play.org/) (v3)
- **Communication**: [Socket.IO](https://socket.io/) (v4)
- **Backend (Required)**: ポーズデータを配信するためのサーバーが必要です（デフォルト設定では `http://100.108.145.71:5000` を参照）。

## ファイル構成

- `index.html`: 体験のメインエントリポイント
- `Untitled-2.html`: 姿勢検出の状態を確認するためのダッシュボード
- `sketch_new.js`: リファクタリングされたメインロジック（推奨）
- `sketch.js`: オリジナルのメインロジック
- `assets/`: アニメーション用スプライトシートや画像
- `lib/`: 必要なライブラリファイル

## セットアップと実行方法

1. **バックエンドサーバーの起動**:
   別リポジトリ（またはローカルのPythonサーバー等）で姿勢検出サーバーを起動し、Socket.IOでデータを配信してください。
   ※接続先IPは `sketch_new.js` の `connectWebSocket` 関数内で変更可能です。

2. **フロントエンドの起動**:
   ローカルサーバー（VSCodeのLive Serverなど）を使用して `index.html` または `Untitled-2.html` を開きます。

3. **操作方法**:
   - カメラの前に立ち、全身または上半身が映るようにしてください。
   - 画面上のオブジェクトに触れたり、手を近づけたりすることでインタラクションが発生します。
